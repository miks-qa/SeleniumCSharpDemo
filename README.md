# Selenium C# Automation Demo :robot:

A Selenium WebDriver automation practice project using C# and NUnit.

## Tech Stack :hammer_and_wrench:

- C#
- Selenium WebDriver
- NUnit
- Visual Studio

## Structure :open_file_folder:

- PageObjects

- Tests
  - Contains automated test scenarios
